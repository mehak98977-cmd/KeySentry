## KeySentry ⌨️
Cybersecurity training is usually ugly and boring. We fixed that.

KeySentry is an interactive, cyberpunk-themed mini game that teaches you how keyloggers work by making you defend against them. No slideshows, no outdated UI. Just real scenarios, neon aesthetics, and a simulated hacker trying to steal your data.



## How it works
It's a 5-level game based on real threats like sketchy WiFi, USB baiting, and phishing.

Read the briefing.
Pick your defenses. (Choose wisely, Incognito mode won't save you here).
Type the target phrase while a keylogger watches.
Check your defenses. You either get a secure terminal log, or you watch your data get sent to a hacker.
We added terminal logs and explanations for every right and wrong answer so you actually learn the "why" instead of just guessing.

## The Vibe
Scanlines, glitch effects, terminal logs, neon accents. If you're teaching people about hackers, it shouldn't look like a corporate PDF.

## Tech Stack
Pure HTML, CSS, and Vanilla JS. Zero frameworks. Zero dependencies. One single file.

## How to run it
Just download the repo and open keysentry.html in your browser. That's it.
